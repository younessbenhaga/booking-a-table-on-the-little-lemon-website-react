# Clone project
```bash
git init
git clone https://github.com/truonganhvu205/little-lemon-website.git
cd little-lemon-website
npm install
```

# Libraries
## React Router
```bash
npm install react-router-dom localforage match-sorter sort-by
```

# Getting Started
```bash
npm start
```
